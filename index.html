БИЛЕТ № 2

1. Теорема Штейнера (без доказательства)

Момент инерции тела $I$ относительно произвольной оси равен сумме момента инерции $I_C$ относительно оси, параллельной данной и проходящей через центр масс тела, и произведения массы тела $m$ на квадрат расстояния $d$ между осями:

$$I = I_C + md^2$$

2. Вынужденные колебания

По второму закону Ньютона для тела массой $m$ под действием квазиупругой силы $-kx$, силы вязкого трения $-r\dot{x}$ и внешней гармонической силы $F_0\cos(\omega t)$:

$$m\ddot{x} = -kx - r\dot{x} + F_0\cos(\omega t)$$

Разделив на $m$ и обозначив $2\beta = r/m$, $\omega_0^2 = k/m$, $f_0 = F_0/m$:

$$\ddot{x} + 2\beta\dot{x} + \omega_0^2 x = f_0\cos(\omega t)$$

Вид решения для установившихся колебаний: $x(t) = A\cos(\omega t - \varphi)$, где $A$ — амплитуда, $\varphi$ — сдвиг фаз.

3. Задача

Дано $n_0 = 360\text{ об/мин} = 6\text{ с}^{-1}$
$N = 150\text{ об}$
$A_\text{тр} = -471\text{ Дж}$
$\omega_k = 0$
Найти: $I,\; M_\text{тр}$
Решение
  1. Начальная угловая скорость: $\omega_0 = 2\pi n_0 = 12\pi\text{ рад/с}$
  2. Угол поворота до остановки: $\Delta\varphi = 2\pi N = 300\pi\text{ рад}$
  3. По теореме об изменении кинетической энергии: $A_\text{тр} = -\dfrac{I\omega_0^2}{2}$, откуда $$I = -\frac{2A_\text{тр}}{\omega_0^2} = \frac{942}{144\pi^2} \approx 0.66\text{ кг·м}^2$$
  4. Момент сил торможения: $$M_\text{тр} = \frac{A_\text{тр}}{\Delta\varphi} = \frac{-471}{300\pi} \approx -0.5\text{ Н·м}$$
Ответ $I \approx 0.66\text{ кг·м}^2$;  $M_\text{тр} \approx -0.5\text{ Н·м}$

БИЛЕТ № 4

1. Момент импульса относительно точки и оси

  • Момент импульса материальной точки относительно полюса $O$: $\vec{L} = [\vec{r}\times\vec{p}]$.
  • Момент импульса относительно оси $z$: $L_z = I_z\omega$.

2. Закон сохранения полной механической энергии

  • Работа консервативной силы: $A = W_{p1} - W_{p2} = -\Delta W_p$.
  • Полная механическая энергия: $E = E_k + W_p$.
  • Закон изменения: $\Delta E = A_\text{неконс}$.
  • Закон сохранения: если $A_\text{неконс} = 0$, то $E = \text{const}$.

3. Задача

Дано $\tau = 5\text{ с}$
$\omega_0 = 20.001\text{ с}^{-1}$
Найти: $\omega,\;\lambda,\;Q$
Решение
  1. Коэффициент затухания: $\beta = 1/\tau = 0.2\text{ с}^{-1}$
  2. Циклическая частота: $$\omega = \sqrt{\omega_0^2 - \beta^2} = \sqrt{400.04 - 0.04} \approx 20\text{ с}^{-1}$$
  3. Логарифмический декремент: $$\lambda = \beta T = 0.2\cdot\frac{2\pi}{20} = 0.02\pi \approx 0.0628$$
  4. Добротность: $Q = \pi/\lambda = 50$
Ответ $\omega = 20\text{ с}^{-1}$;  $\lambda \approx 0.0628$;  $Q = 50$

БИЛЕТ № 6

1. Полное, тангенциальное и нормальное ускорение

  • Полное: $\vec{a} = \vec{a}_\tau + \vec{a}_n$
  • Тангенциальное (быстрота изменения модуля скорости): $a_\tau = dv/dt$
  • Нормальное (быстрота изменения направления): $a_n = v^2/R$
  • Модуль: $a = \sqrt{a_\tau^2 + a_n^2}$

2. Потенциальная энергия в поле силы тяжести

  • В общем случае: $W_p = -G\dfrac{mM}{r}$
  • В однородном поле (вблизи поверхности): $W_p = mgh$

3. Задача

Дано $m_\text{диска} = 2m$, $r_\text{диска} = R$, $\omega_\text{нач} = \omega_0$
$m_\text{снаряда} = m$, $r_\text{снаряда} = R/2$, $V_\text{снаряда} = 4\omega_0 R$
Найти: $\omega_x$
Решение
  1. Применяем ЗСМ импульса: $L_\text{до} = L_\text{после}$
  2. До выстрела: $I_д = mR^2$, $I_с = 0.25mR^2$, $L_\text{до} = 1.25mR^2\omega_0$

БИЛЕТ № 8

1. Угловая скорость и ускорение твёрдого тела

  • Угловая скорость $\omega = d\varphi/dt$, вектор $\vec{\omega}$ — вдоль оси по правилу правого винта.
  • Угловое ускорение $\varepsilon = d\omega/dt$.
  • $v=\omega R$;  $a_\tau = \varepsilon R$;  $a_n = \omega^2 R$;  $a = R\sqrt{\varepsilon^2+\omega^4}$

2. Момент импульса. Уравнение моментов. ЗСМ импульса

  • Момент импульса точки: $\vec{L} = [\vec{r}\times m\vec{v}]$
  • Уравнение моментов: $\dfrac{d\vec{L}}{dt} = \vec{M}_\text{внеш}$
  • ЗСМ импульса: если $\vec{M}_\text{внеш} = 0$, то $\vec{L} = \text{const}$

3. Задача — цепочка на столе

Дано Свешивающаяся часть $l_1 = L/5$ — начало скольжения
Найти: $\mu$
Решение

Масса свешивающейся части: $m_1 = M/5$. Масса на столе: $m_2 = 4M/5$.

Условие начала скольжения ($F_\text{тяж} = F_\text{тр}$):

$$\frac{M}{5}g = \mu\cdot\frac{4M}{5}g \;\Rightarrow\; \mu = \frac{1}{4}$$
Ответ$\mu = 0.25$

БИЛЕТ № 10

1. Момент инерции. Момент инерции однородного диска

Момент инерции — мера инертности при вращении: $I = \int r^2\,dm$.

Для однородного диска (цилиндра) относительно оси симметрии: $$I = \frac{1}{2}mR^2$$

2. Потенциальная энергия силы упругости

Закон Гука: $F = -kx$. Работа: $A = -\Delta W_p$. Следовательно:

$$W_p = \frac{kx^2}{2}$$

3. Задача

Дано $T = 5\text{ с}$;  $A_0/A_t = 20$;  $t = 500\text{ с}$
Найти: $\beta,\;\lambda,\;Q$
Решение
  1. Из $A_0/A_t = e^{\beta t}$: $$\beta = \frac{\ln 20}{t} = \frac{2.996}{500} \approx 0.006\text{ с}^{-1}$$
  2. $\lambda = \beta T = 0.006\cdot5 \approx 0.03$
  3. $Q = \pi/\lambda \approx 104.8$

БИЛЕТ № 12

1. Работа. Кинетическая энергия. Теорема о кинетической энергии

  • Работа: $A = \int\vec{F}\cdot d\vec{r}$
  • Кинетическая энергия: $W_k = mv^2/2$; при вращении $W_k = I\omega^2/2$
  • Теорема: $A = \Delta W_k = W_{k2} - W_{k1}$

2. Виды равновесия. Квазиупругая сила. Уравнение гармонических колебаний

  • Устойчивое — $W_p$ минимальна; Неустойчивое — $W_p$ максимальна; Безразличное — $W_p = \text{const}$
  • Квазиупругая сила: $F = -kx$
  • Уравнение: $\ddot{x}+\omega_0^2 x = 0$, где $\omega_0^2 = k/m$
  • Решение: $x(t) = A\cos(\omega_0 t + \varphi_0)$

3. Задача — блок (диск) и тело

Дано $m = 10\text{ кг}$ (масса блока = масса тела);  $F = 150\text{ Н}$
Блок — сплошной диск
Найти: $a$
Решение
  1. Для тела: $ma = F - T$  (1)
  2. Для блока: $I\varepsilon = TR$, $I=\tfrac{1}{2}mR^2$, $\varepsilon=a/R$ → $T = \tfrac{1}{2}ma$  (2)
  3. Подставляем (2) в (1): $ma = F - \tfrac{1}{2}ma$ → $\tfrac{3}{2}ma = F$ → $$a = \frac{2F}{3m} = \frac{300}{30} = 10\text{ м/с}^2$$
Ответ$a = 10\text{ м/с}^2$

БИЛЕТ № 13

1. Кинетическая энергия вращающегося тела

Разбиваем тело на элементарные массы $dm_i$ с расстояниями $r_i$ до оси, $v_i = \omega r_i$:

$$W_k = \sum\frac{dm_i(\omega r_i)^2}{2} = \frac{\omega^2}{2}\sum dm_i r_i^2 = \frac{I\omega^2}{2}$$

2. Векторная диаграмма. Сложение гармонических колебаний одной частоты

Колебания $x_1 = A_1\cos(\omega t+\varphi_1)$, $x_2 = A_2\cos(\omega t+\varphi_2)$ представляются вращающимися векторами. Результирующее $x = A\cos(\omega t+\varphi)$:

$$A = \sqrt{A_1^2 + A_2^2 + 2A_1 A_2\cos(\varphi_2-\varphi_1)}$$

$$\tan\varphi = \frac{A_1\sin\varphi_1 + A_2\sin\varphi_2}{A_1\cos\varphi_1 + A_2\cos\varphi_2}$$

3. Задача — физический маятник (диск)

Дано $R = 0.5\text{ м}$;  $d = 0.25\text{ м}$;  $\lambda = 1$
Найти: $T_0,\;\nu_0,\;\nu'$
Решение
  1. $I = \tfrac{1}{2}mR^2 + md^2 = m(\tfrac{R^2}{2}+d^2)$
  2. $$T_0 = 2\pi\sqrt{\frac{R^2/2+d^2}{gd}} = 2\pi\sqrt{\frac{0.1875}{2.45}} \approx 1.74\text{ с}$$
  3. $\nu_0 = 1/T_0 \approx 0.57\text{ Гц}$
  4. $$\nu' = \frac{\nu_0}{\sqrt{1+\lambda^2/(4\pi^2)}} \approx \frac{0.57}{1.012} \approx 0.56\text{ Гц}$$
Ответ $T_0 \approx 1.74\text{ с}$;  $\nu_0 \approx 0.57\text{ Гц}$;  $\nu' \approx 0.56\text{ Гц}$

БИЛЕТ № 17

1. Перемещение, скорость, ускорение

  • Перемещение: $\Delta\vec{r} = \vec{r}(t+\Delta t)-\vec{r}(t)$
  • Скорость: $\vec{v} = \lim_{\Delta t\to0}\dfrac{\Delta\vec{r}}{\Delta t} = \dfrac{d\vec{r}}{dt}$
  • Ускорение: $\vec{a} = \dfrac{d\vec{v}}{dt} = \dfrac{d^2\vec{r}}{dt^2}$

2. Потенциальная энергия в поле тяжести (вывод)

  • Общий случай (поле тяготения): $$W_p = -G\frac{mM}{r}$$

Ось — через центр стержня
Найти: $\nu_0$
Решение
  1. Координата ЦМ системы: $d_c = L/4 + R/2 = 0.025+0.025 = 0.05\text{ м}$

  • $a_n = v^2/R$ — изменение направления скорости, к центру кривизны

2. Собственная частота малых колебаний физического маятника (вывод)

При отклонении на угол $\varphi$: $I\ddot{\varphi} = -mgl\sin\varphi$. Для малых углов $\sin\varphi\approx\varphi$:

3. Задача — абсолютно неупругий удар по стержню

Дано Стержень: $M=2.7\text{ кг}$, $L=1\text{ м}$
Шарик: $m=0.1\text{ кг}$, $v=10\text{ м/с}$
Найти: $\omega$ после удара
Решение
  1. ЗСМ импульса относительно оси подвеса: $mvL = I_\text{сист}\cdot\omega$
  2. $I_\text{сист} = \tfrac{1}{3}ML^2 + mL^2 = \left(\tfrac{M}{3}+m\right)L^2 = 1.0\text{ кг·м}^2$
  3. $$\omega = \frac{mv}{\left(\tfrac{M}{3}+m\right)L} = \frac{0.1\cdot10}{1.0\cdot1} = 1\text{ рад/с}$$
Ответ$\omega = 1\text{ рад/с}$
